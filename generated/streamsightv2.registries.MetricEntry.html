streamsightv2.registries.MetricEntry

class streamsightv2.registries.MetricEntry(name: str, K: int | None = None)

Bases: NamedTuple

Entry for the metric registry.

The intended use of this class is to store the name of the metric and the top K value for the metric specified by the user. Mainly this will happen during the building phase of the evaluator pipeline in Builder.

Parameters:

  • name (str) – Name of the algorithm

  • K (int, optional) – Top K value for the metric.
